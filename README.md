# Simulation-Based Inference for Drift-Diffusion Models of Decision-Making

This project implements a modular simulation-based inference pipeline for recovering latent cognitive decision parameters from reaction-time and choice data generated by a drift-diffusion model (DDM).

The goal is to demonstrate how Bayesian machine learning methods can infer hidden decision-making parameters when the likelihood is difficult to evaluate analytically. The project connects simulation-based inference, latent variable modelling, behavioural data analysis, and uncertainty quantification.

## Project Motivation

Many cognitive and behavioural models describe hidden mental processes that cannot be directly observed. In the drift-diffusion model, decisions are modelled as noisy evidence accumulation over time. The observed data consist of reaction times and binary choices, while the underlying parameters represent latent cognitive quantities such as evidence accumulation speed, decision caution, and non-decision time.

This project uses neural posterior estimation to infer these latent parameters from simulated behavioural data.

## Model

The drift-diffusion model simulates a decision process in which noisy evidence accumulates until it reaches one of two decision boundaries.

The inferred parameters are:

| Parameter | Interpretation |
|---|---|
| `drift_rate` | Strength and direction of evidence accumulation |
| `boundary` | Decision threshold or caution level |
| `non_decision_time` | Perceptual and motor delay outside the decision process |

The simulator generates:

- reaction times
- binary choices
- summary statistics used for inference

## Method

The project uses simulation-based inference with neural posterior estimation.

The pipeline is:

1. Define prior distributions over DDM parameters.
2. Simulate reaction-time and choice data from sampled parameters.
3. Convert behavioural data into fixed-size summary statistics.
4. Train a neural posterior estimator using simulated parameter-data pairs.
5. Infer posterior distributions over latent parameters for observed data.
6. Evaluate parameter recovery and posterior uncertainty.
