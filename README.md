# Data

This project uses simulated behavioural data generated from a drift-diffusion model.

The file `simulated_observed_data.csv` contains synthetic reaction-time and binary-choice observations generated using the following latent parameters:

- drift rate: 1.20
- decision boundary: 1.00
- non-decision time: 0.25

No external human-subject dataset is used. The purpose of the simulated dataset is to demonstrate simulation-based inference for recovering latent cognitive decision parameters.
